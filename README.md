# Virtual-Control-using-Hand-Gestures

A computer vision project that enables virtual control of computer functions using real-time hand gestures captured via webcam. Actions include increasing/decreasing volume, scrolling, tab switching, and exitingâ€”all hands-free.

## Features
- Real-time hand gesture recognition
- Control volume, tabs, scrolling, and more
- Touchless interface using camera

## Tech Stack
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## How to Run
1. Clone the repo
2. Install dependencies: 
pip install -r requirements.txt
3. Run the app: 
python main.py
